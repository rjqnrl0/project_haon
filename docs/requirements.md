# V-Suitcase — 프로젝트 요구사항 정의서

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| **서비스명** | V-Suitcase |
| **유형** | AI 기반 초개인화 라이프스타일 시뮬레이션 플랫폼 |
| **한 줄 요약** | 나의 모습과 나의 공간에 원하는 패션을 자유롭게 입혀보는 AI 가상 시뮬레이션 서비스 |

## 2. 핵심 기능 (Core Features)

### 2.1 AI 스마트 가상 피팅 (AI Smart Virtual Fitting)
- 사용자의 전신 사진을 기반으로 옷·가방·액세서리 이미지를 합성하여 실제 착용한 듯한 피팅 결과 생성
- **UX 플로우**: 사진 1장 업로드 → 관심 상품 선택 → AI가 자연스러운 핏·질감·그림자를 반영한 피팅 이미지 제공

### 2.2 커스텀 배경 제너레이터 (Custom Background Generator)
- 피팅 완료 후 배경을 원하는 장소나 분위기로 변경
- **입력 방식**
  - 텍스트 프롬프트 (예: "저녁 노을이 지는 제주도 해변")
  - 이미지 직접 업로드

### 2.3 AI 공간 스타일링 (확장 기능 — AI My Room Interior)
- 거주 공간 사진 업로드 후 가전·가구 가상 배치
- 조명·원근감 분석을 통한 자연스러운 인테리어 프리뷰 제공

## 3. 세부 기능 요구사항

### 3.1 가상 피팅 엔진 (Core Engine)

| ID | 요구사항 | 설명 |
|----|----------|------|
| REQ-01 | 자동 마스킹 | 옷 카테고리에 맞춰 인물 사진의 해당 영역 자동 분할 (Segmentation) |
| REQ-02 | 배경 상황 최적화 | 여행지 배경에 맞는 광원·그림자 계산으로 자연스러운 합성 수행 |
| REQ-03 | 텍스처 보존 | 실크·면 등 고유 재질이 합성 후에도 유지되도록 AI 파라미터 최적화 |

### 3.2 지능형 추천 및 분석 (AI Analysis)

| ID | 요구사항 | 설명 |
|----|----------|------|
| REQ-04 | 체형별 사이즈 조언 | Claude 3.5가 어깨선·허리선을 분석해 사이즈업/다운 추천 메시지 제공 |
| REQ-05 | 날씨 기반 코디 | 여행지 실시간 날씨 API 연동, 날씨에 맞는 코디 제안 |

### 3.3 시스템 관리 및 인프라 (Admin & Infra)

| ID | 요구사항 | 설명 |
|----|----------|------|
| REQ-06 | 상품 데이터 파이프라인 | 제휴 쇼핑몰 신규 의류 등록 시 자동 배경 제거(누끼) 및 학습용 이미지 생성 스케줄러 구축 |
| REQ-07 | 로그 및 모니터링 | CloudWatch로 AI 추론 실패율·사용자 피드백 통계 대시보드 제공 |
