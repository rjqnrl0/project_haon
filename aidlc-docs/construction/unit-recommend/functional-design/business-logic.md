# Business Logic — unit-recommend

## 1. RecommendService (BC-05)

### 1.1 사이즈 추천 (US-10)

**플로우**:
1. 피팅 결과 이미지 + 의류 정보 수신
2. Phase 1: 하드코딩된 추천 텍스트 반환 (AI 응답 형식)
3. Phase 2: Claude Opus (Bedrock) 호출 → 체형 분석 + 사이즈 추천 생성

**Phase 1 Mock 구현**:
- 의류 카테고리에 따른 미리 작성된 추천 텍스트 반환
- AI 응답 형식 미리보기 목적 (UX 검증용)
- 사이즈 표, 핏 조언, 스타일링 팁 포함

**Phase 1 Mock 응답 템플릿**:
```json
{
  "sizeRecommendation": {
    "recommendedSize": "M",
    "sizeChart": { "S": "...", "M": "...", "L": "..." },
    "fitAdvice": "이 상의는 레귤러핏으로, 평소 사이즈를 추천드립니다.",
    "stylingTip": "슬림핏 하의와 매치하면 균형잡힌 실루엣을 연출할 수 있습니다."
  }
}
```

**Phase 2 전환**:
- Mock 텍스트 → Claude Opus via Bedrock
- 프롬프트: 전신 이미지 분석 + 의류 메타데이터 기반 개인화 추천
- 응답 구조는 Phase 1과 동일 (인터페이스 유지)

**비즈니스 규칙**:
- 피팅 결과가 존재해야 추천 가능
- 카테고리별 추천 (상의/하의/원피스/아우터 각각)
- 추천 결과 캐싱: 동일 피팅 결과에 대해 1회만 생성

### 1.2 날씨 코디 추천 (US-11)

**플로우**:
1. 사용자가 여행지 입력 (자동완성 지원)
2. OpenWeatherMap API 호출 → 날씨 데이터 조회
3. Phase 1: 날씨 데이터 + 하드코딩 코디 추천 텍스트 반환
4. Phase 2: 날씨 데이터 + Claude Opus → 개인화 코디 추천 생성

**OpenWeatherMap 연동**:
- Endpoint: `GET /data/2.5/forecast`
- Parameters: `q={city}`, `cnt=5` (5일 예보)
- 응답에서 추출: 기온, 체감온도, 날씨 상태, 강수 확률, 습도

**Phase 1 코디 추천 로직**:
```
기온 기반 분류:
- 30°C 이상: "더운 날씨" 템플릿
- 20~29°C: "선선한 날씨" 템플릿
- 10~19°C: "쌀쌀한 날씨" 템플릿
- 10°C 미만: "추운 날씨" 템플릿

추가 조건:
- 강수확률 50% 이상: 우비/방수 아우터 추천 추가
- 습도 80% 이상: 통기성 좋은 소재 추천 추가
```

**Phase 2 전환**:
- 룰 기반 분류 → Claude Opus 개인화 추천
- 날씨 데이터 + 사용자 의류 정보 + 여행 기간 종합 분석

**비즈니스 규칙**:
- 도시명 유효성: OpenWeatherMap 응답 성공 여부로 판단
- 검색 실패 시: "도시를 찾을 수 없습니다" 에러
- 예보 기간: 최대 5일
- 여행 날짜 미지정 시 내일~5일 기본 조회

### 1.3 여행지 자동완성

**구현 방식**:
- 프론트엔드에 인기 도시 목록 내장 (JSON)
- 입력 시 클라이언트 사이드 필터링 (서버 호출 없음)
- 한국어/영어 도시명 모두 매칭

**인기 도시 목록 (초기)**:
- 국내: 서울, 부산, 제주, 강릉, 경주, 전주, 여수
- 해외: 도쿄, 오사카, 방콕, 싱가포르, 파리, 런던, 뉴욕, 하와이, 발리, 다낭

## 2. 추천 결과 구조 (상세 카드)

### 사이즈 추천 카드

| 섹션 | 내용 |
|------|------|
| 추천 사이즈 | M (메인 추천값) |
| 사이즈 표 | S/M/L/XL 별 신체 치수 범위 |
| 핏 조언 | 핏 유형 설명 + 사이즈 선택 이유 |
| 스타일링 팁 | 매치 추천 의류 + 스타일링 조언 |

### 날씨 코디 카드

| 섹션 | 내용 |
|------|------|
| 여행지 날씨 요약 | 기온, 날씨 상태, 강수 확률 (5일) |
| 코디 추천 | 날씨에 맞는 의류 조합 추천 |
| 필수 아이템 | 반드시 챙겨야 할 아이템 |
| 추가 팁 | 현지 스타일 팁, 주의사항 |
